# Quick Start for Replit Setup (30 seconds)

## If you're in a hurry, do this:

### 1. Create Replit Account
- Go to https://replit.com → Sign up (free)

### 2. Create New Replit
- Click Create → New Replit
- Select Python
- Name it: `portfolio-backend`

### 3. Copy Backend Code
- Create new file `main.py`
- Copy ALL code from `scripts/replit_main.py` into it

### 4. Copy Dependencies
- Create new file `requirements.txt`
- Copy ALL code from `scripts/replit_requirements.txt` into it

### 5. Run It
- Click green "Run" button
- Wait for dependencies to install (~30 seconds)

### 6. Get Your URL
- Look at top-right corner
- Copy the URL (like: `https://portfolio-backend.yourname.repl.co`)

### 7. Connect to Frontend
- Open your portfolio website
- Press S → H → E (keys)
- Click Edit next to "Backend: OFFLINE"
- Paste your Replit URL
- Save & Login (admin / admin123)

### 8. Done! Upload Files!

---

### Keep Backend Online
- Keep Replit tab open in browser
- Backend stays online 24/7 when tab is active
- Close tab = backend offline (but files are saved)

---

**That's it!** You now have a working backend. Read `REPLIT_SETUP_GUIDE.md` for detailed instructions.
